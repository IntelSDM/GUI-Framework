# GUI-Framework
Object-Oriented GUI framework to create a GUI using form priority and a stack to manage the drawing list.
Effectively allows objects of the GUI to manage each other.
<br> 
Includes:
1. Forms
2. Tabs
3. Buttons
4. TextBoxes
5. Toggles
6. Dynamic Sliders
7. Labels
